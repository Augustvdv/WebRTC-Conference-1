WebRTC-Conference

WebRTC-Conference is a progressive web application which provides users participating in a conference the ability to listen to a session done in a foreign language, in a language familiar to the user, by tuning into a channel which broadcasts the same session in the language of your choice.

Setup

Prerequisites

Install polymer-cli:

npm install -g polymer-cli

bower install

Start the development server

This command serves the app at http://localhost:8080 and provides basic URL routing for the app:

polymer serve --open
